# express-test-app

## Install dependencies

```bash
cd express-test-app
cd client
npm install
cd ..
npm install
```

## Start the development server

```bash
npm run dev
```

## Run tests

```bash
npm run test
```

## Build production bundle

```bash
npm run build
```

## Run production server

```bash
NODE_ENV="production" node ./dist/index.js
```
